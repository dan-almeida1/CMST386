Describe how search engines work:

Search engines search the web to find web pages. The search results are presented in order of relevance to the search query.
Search engines do this through crawling, indexing, and ranking.
Crawling is when search engines find and examine pieces of publicly accessible content.
Indexing is when they categorize each piece of content.
Ranking is when they decide what content is most relevant to visitors

Explain the importance of keywords and descriptions as meta tags

Meta tags are one of the ways a website can communicate with a search engine. Meta description tags offer a description of the page, which appears on the search results. Keywords offer to search engines a better understanding about what your website is about.

List the four elements that comprise writing relevant content links:

The four elements that comprise writing relevant content links are:

  • Technical SEO - Make sure crawlers can properly crawl and index your website by keeping your website organized and using the appropriate meta tags.
  • Content - Your content's relevance to the search term.
  • On-Site SEO - The optimization of your HTML
  • Off-Site SEO - How you build reputation with search engines off of your website
